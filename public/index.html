What is Fraud Arret

Fraud Arret helps Businessmen to protect them from malicious fraudsters. It screens all orders transacted using credit cards, PayPal, and so on for online frauds (also known as payment frauds or CNP frauds). As a result, it increases e-commerce profits by reducing chargeback, improving operation efficiency and increasing revenue.Businessmen can investigate all complex, high-risk orders in a simple way by using administrative interface.

Why Fraud Arret

FraudArret comprehensively validates all elements such as geolocation, proxy, email, credit card, transaction velocity and others to unveil fraud orders accurately. It's more than the fraud detection solution, as we offer users more than 40 validation rules to custom make their own validation logic. In addition, we have millions of blacklist records contributed by our global merchant community to further protect your online business from malicious fraudsters. And, the list is growing.

Features

How does the Fraud Arret Works?

FraudArret accepts online transactions data via its web programming interface (FraudArret API). The FraudArret analytic engine analyzes transactions parameters and reports fraud analysis in less than a second. Businessmen can immediately decide next action based on fraud distribution score or custom rules by conditions. Further analysis of transactions is also available instantly in the administrative interface.

How does the Fraud Arret algo Work

FraudArret has evolved from a blacklist based algorithm to machine learning-based algorithm in fraud detection. Businessmen can enhance the algorithm by accepting or rejecting any real-life transactions to improve the classification process.
We also perform screening of IP address, email address, credit cards and devices based on moderated crowd-sourcing data. These high-quality blacklists can catch fraud occurrence effectively.

Key Features

FraudArret runs a thorough validations on an order transaction to effectively detect a malicious fraud. Below are the key features and type of fraud validation performed:

  • Fraud analysis and score
  • IP address geolocation using IP2Location

  • Email address validation using MailBoxValidator
  • Credit card issuing bank validation
  • Transaction velocity validation
  • Device transaction validation
  • Blacklist validation
  • High risk username & password validation
  • Export controlled country validation
  • Malware exploit validation
  • Custom rules trigger
  • FraudArret Merchant Network
  • FraudArret Merchant Administrative Interface
  • Email notification of fraud orders
  • Mobile app notification of fraud orders
  • Multi factor authentication using SMS
